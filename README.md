# Swapsicle Contract Addresses

This repository maintains a list of addresses useful for interacting with the Swapsicle protocol.

These addresses may change upon smart contract re-deployment (especially on testnets). This repository will be updated with the new addresses as they change.

## Usage

This repository is public to facilitate its integration as a Git submodule. For more information on Git submodules, refer to the [Git documentation](https://git-scm.com/book/en/v2/Git-Tools-Submodules).
